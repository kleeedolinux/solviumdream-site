socket.go

Uma biblioteca Go de alto desempenho para comunicação em tempo real com WebSocket e long-polling, com processos distribuídos inspirados em Erlang, árvores de supervisão autorrecuperáveis e mensagens baseadas em salas. A high-performance Go library for real-time communication with WebSocket and long-polling, featuring Erlang-inspired distributed processes, self-healing supervision trees, and room-based messaging.

Go / WebSocket

gorilix

Um framework leve de modelo de ator tolerante a falhas para Go. Inspirado no Erlang/OTP, o Gorilix permite escalabilidade perfeita e recuperação robusta de erros em sistemas distribuídos. A lightweight, fault-tolerant actor model framework for Go. Inspired by Erlang/OTP, Gorilix enables seamless scalability and robust error recovery in distributed systems.

Go / Actor Model

Go on Airplanes

Um framework web Go leve e de alto desempenho com roteamento baseado em arquivos HTML onde páginas no diretório do aplicativo se tornam automaticamente rotas. A lightweight, high-performance Go web framework featuring HTML file-based routing where pages in the app directory automatically become routes.

Go / Web Framework
